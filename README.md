## Featured

### Npj-writing

`Npj-writing` is a Codex skill for `npj Computational Materials` style manuscript writing and revision.

It focuses on:

- materials-informatics and high-throughput molecular screening manuscripts
- applicability-domain, OOD, uncertainty, and disagreement-aware claim calibration
- mechanism-aware two-photon photoinitiator discovery
- ML and quantum-mechanical workflow reporting
- restrained NPJ Article structure for title, abstract, introduction, Results, Discussion, Methods, and figure narrative

Repository: https://github.com/Shiruo-Yang/Npj-writing

## Current writing principle

Computational screening should be presented as auditable candidate prioritization unless quantum-mechanical and experimental evidence supports stronger photoinitiator claims.
